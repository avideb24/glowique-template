Introduction

Vida Template is a modern, responsive Next.js template built with React and TailwindCSS. It provides a solid foundation for building beautiful web applications with a focus on performance and user experience.

Installation

# Clone the repository
git clone [repository-url]

# Navigate to project directory
cd vida-template

# Install dependencies
npm install

Getting Started

To start the development server:

npm run dev

The application will be available at http://localhost:3000

Customization

Theme Configuration

You can customize the theme by modifying the following files:

  • tailwind.config.mjs - For TailwindCSS configuration
  • src/styles/ - For custom CSS styles

Deployment

To build and deploy the application:

# Build the application
npm run build

# Start the production server
npm start

Dependencies

Main Dependencies

  • Next.js 15.1.6
  • React 19.0.0
  • TailwindCSS 3.4.1
  • Ant Design 5.23.3
  • Axios 1.8.2
  • React Icons 5.5.0
  • Swiper 11.2.2

Troubleshooting

Common Issues

Build Errors

If you encounter build errors, try the following:

  • Clear the Next.js cache: rm -rf .next
  • Reinstall dependencies: rm -rf node_modules && npm install
